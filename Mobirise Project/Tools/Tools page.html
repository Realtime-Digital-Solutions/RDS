The GitHub trail doesn’t necessarily reflect the effort of everyone in the group. We assigned ourselves each a section of the assignment to complete for the webpage and then created that page in a program called Mobirise. We based this off our template from assignment 2. We did most of our editing in Mobirise and then put the code into GitHub for publishing and final tweaking. The GitHub trail does, however, highlight the teams’ ability to problem-solve and work together to fix coding errors, as with assignment 2.

For this assignment we worked primarily within MS Teams to collaborate on our project and host regular meetings. We would meet twice per week on Thursdays and Sundays and schedule this in using the MS Teams calendar. We took turns hosting the meetings and used the files sections to create a library for our agendas, recordings, and actions. When utilised the group chat feature to communicate in between meetings.
